isFocus

PURPOSE

focusNum : Computes how in focus an image is

SYNOPSIS

function focusNum = isFocus( p1 )

DESCRIPTION 

 focusNum : Computes how in focus an image is
 by multiplying the fourier transform with the complex conjugate.

 INPUT :
      p1 : fourier transform of image

 OUTPUT :
       focusNum : error value
